# Mock Product Idea

Build a team activity digest that gives workspace admins a weekly view of the
most important collaboration changes.

Known context:

- Admins want a concise digest they can scan in under two minutes.
- The digest should include new comments, resolved threads, label changes, and
  unusually active sites.
- Users should be able to opt out per workspace.
- Email delivery should use the existing email architecture.
- The first release can be admin-only and does not need custom schedules.

Open ambiguity:

- The exact email layout is not approved.
- The threshold for "unusually active" needs a recommended default.

Expected planning outcome:

- A Linear Project or PRD that captures goals, non-goals, user stories,
  implementation decisions, verification expectations, risks, and HITL items.
